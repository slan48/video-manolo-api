# Video manolo

API to allow client to get movies, manage users and do reservations.

## Structure

The current structure of the project is:

```
__test__/        
config/
models/
routes/
utils/
```


## Install

Via npm

``` bash
$ npm install
```

Then you have to create ``.env`` file following ``.env.example`` file structure before run project

## Usage

#### Initial seed 
``` javascript
npm run seed
```

#### Development
``` javascript
npm run dev
```

#### Production
``` javascript
npm run start
```

## Testing

``` bash
$ npm run test
```
